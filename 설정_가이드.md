# Kanade 클라우드 설정 가이드 (Firebase + GitHub Pages)

전부 **무료**이고, 처음부터 끝까지 약 20분 걸립니다.
순서: ①Firebase 만들기 → ②설정값 붙여넣기 → ③GitHub Pages에 올리기 → ④마지막 연결

---

## ① Firebase 만들기 (구글 로그인 + 클라우드 DB)

1. https://console.firebase.google.com 접속 → 구글 계정으로 로그인
2. **"프로젝트 만들기"** → 이름 `kanade` 입력 → **Google 애널리틱스는 끄기** → 만들기
3. 왼쪽 메뉴 **빌드 > Authentication** → **시작하기**
   → "로그인 방법" 탭에서 **Google** 클릭 → **사용 설정** 켜기 → 지원 이메일 선택 → 저장
4. 왼쪽 메뉴 **빌드 > Firestore Database** → **데이터베이스 만들기**
   → 위치: **asia-northeast3 (서울)** → **프로덕션 모드에서 시작** → 만들기
5. 만들어지면 위쪽 **"규칙"** 탭 클릭 → 내용 전부 지우고 아래를 붙여넣기 → **게시**

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
```

(뜻: 로그인한 본인만 자기 기록을 읽고 쓸 수 있음. 남의 기록은 절대 못 봄)

6. 왼쪽 위 **⚙(톱니) > 프로젝트 설정** → "일반" 탭 아래로 내려 **내 앱** → **웹 아이콘 `</>`** 클릭
   → 앱 닉네임 `kanade` → (Firebase 호스팅 체크 안 함) → **앱 등록**
7. 화면에 나오는 코드 중 `const firebaseConfig = { ... };` 부분을 **복사**

## ② 설정값 붙여넣기

복사한 값을 **Claude에게 그대로 붙여넣어 주세요** — 제가 파일에 넣고 동작 확인까지 해드립니다.

(직접 하려면: `가나_암기카드.html`을 메모장으로 열고 `FIREBASE_CONFIG` 부분의
"여기에_붙여넣으세요" 6개를 복사한 값으로 교체)

## ③ GitHub Pages에 올리기 (무료 영구 주소)

1. https://github.com 가입 (아이디가 주소가 됩니다: `아이디.github.io`)
2. 우상단 **+ > New repository** → 이름 `kanade` → **Public** → **Create repository**
3. 가운데 **"uploading an existing file"** 링크 클릭 → 폴더에서 이 **3개 파일만** 드래그:
   - `가나_암기카드.html`
   - `index.html`
   - `icon-180.png`
   - ⚠️ `server.ps1` / `.bat` / `cloudflared.exe` / 이 가이드는 올리지 마세요
4. 아래 **Commit changes** 버튼
5. **Settings > Pages** → Branch: **main**, 폴더 **/(root)** → **Save**
6. 1~2분 후 `https://아이디.github.io/kanade/` 접속 → 앱이 뜨면 성공

## ④ 마지막 연결 (이거 빼먹으면 구글 로그인이 안 돼요!)

1. Firebase 콘솔 → **Authentication > 설정 > 승인된 도메인** → **도메인 추가**
   → `아이디.github.io` 입력 → 추가
2. 아이폰 사파리에서 `https://아이디.github.io/kanade/` 열기
   → **공유 버튼 → 홈 화면에 추가** → カ 아이콘 앱처럼 사용

## 자주 묻는 것

- **firebaseConfig는 공개돼도 되나요?** 네. 비밀번호가 아니라 "어느 프로젝트인지" 식별자예요.
  데이터 보안은 ⑤번에서 넣은 규칙이 지킵니다.
- **무료 한도는?** 하루 읽기 5만 회 / 쓰기 2만 회. 혼자 쓰면 평생 무료 범위입니다.
- **파일을 수정하면?** GitHub 저장소에서 파일을 다시 업로드(같은 이름이면 덮어쓰기)하면
  1~2분 뒤 사이트에 반영됩니다.
- **기존 기록은?** 첫 구글 로그인 때 이 기기의 게스트/예전 기록을 자동으로 클라우드로 가져갑니다.
